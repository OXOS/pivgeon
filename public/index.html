Pivgeon: Pivotal Tracker email integration

CC emails to Pivgeon and have stories created automatically in Pivotal Tracker.

Create your first story

1
Send Pivgeon your Pivotal Tracker API token:

To create account in Pivgeon send an email containing API Token you want to use for this email address.

Show more details Next time when you send Pivgeon emails from your address it'll use this API token to access projects.

2
Activate your account:

You should receive a confirmation email. Click the activation link.

3
Send an email to the person you want to create story for.

Just send an email to the person you want to create story for and CC to [project_name]@pivgeon.com.

Show more details If your project name contains spaces, don't include them in the address. For example, if your project is "Web Project", CC your emails to webproject@pivgeon.com. It's also case insensitive. Story description is filled in from email body. Pivgeon will look at To: field of email it was CC-ed and will assign the story to project member with this email address.
