Piv Geon

Pivotal Tracker Integration

Introduction

PivGeon is Pivotal Tracker integration that allows you to create stories directly from emails.

Only 3 steps to create new story

1
Create your account assigned to your pivotal API Token:

To create new account you need to send an email to PivGeon address "pivgeon@pivgeon.com" containing your API Token in subject.

2
Activate your account:

Freshly created account is not ready to use and you need to activate it. PivGeon will send you an email with link that you need to click in order to make your account finally activated.

3
Send an email to the person you want to create story for.

Just send an email to the person you want to create story for and add 'cc' to pivgeon@pivgeon.com.
