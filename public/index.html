Piv Geon

Pivotal Tracker Integration

Introduction

PivGeon is Pivotal Tracker integration that allows you to create stories directly from emails.

Only 3 steps to create new story

1
Create your account assigned to your pivotal API Token:

In order to create new account you need to send an email to PivGeon address "pivgeon@pivgeon.com" containing your API Token in subject.

Show more details You can create many accounts for one api token. You can create only one account for specific email address. Api token you have chosen determines who is set as story creator.

2
Activate your account:

Freshly created account is not ready to use and you need to activate it. PivGeon will send you an email with link that you need to click in order to make your account finally activated.

3
Send an email to the person you want to create story for.

Just send an email to the person you want to create story for and add 'cc' to pivgeon@pivgeon.com.

Show more details In order to specify project name the email subject has to contain project name enclosed in square brackets, for example email subject may look like this "[A new project] This is first story". Project name is case insensitive so there is no difference whether you type "A New Project" or "a new project". Story description is filled in from email body. By default story type is set to 'chore'.
